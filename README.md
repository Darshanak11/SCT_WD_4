## ✨ Features

- 🗂️ **Multiple Lists** — create, switch between, and delete custom task lists
- ➕ **Rich Task Creation** — add title, description, due date, due time, and priority
- 🎯 **Priority Levels** — mark tasks as Low, Medium, or High priority
- ✅ **Task Completion** — check off tasks and track your progress
- ✏️ **Inline Editing** — edit any task via a clean modal without losing context
- 🔍 **Filter Tasks** — view All, Active, or Completed tasks per list
- 📊 **Live Stats** — total and completed task counts update in real time
- 💾 **localStorage Persistence** — your data survives page refreshes and browser restarts
- 📱 **Fully Responsive** — works seamlessly on mobile, tablet, and desktop
- 🎞️ **Smooth Animations** — slide-in transitions on every interaction

## 🎮 How to Use

### Managing Lists
| Action | How |
|--------|-----|
| **Create a list** | Click **New List** in the sidebar |
| **Switch lists** | Click any list name in the sidebar |
| **Delete a list** | Hover over a list and click **Remove** |

### Managing Tasks
| Action | How |
|--------|-----|
| **Add a task** | Fill in the form and click **Add Task** |
| **Complete a task** | Check the checkbox on the task card |
| **Edit a task** | Click **Edit** on any task card |
| **Delete a task** | Click **Delete** on any task card |
| **Filter tasks** | Use the **All / Active / Completed** buttons |
